# student.performance.data.nalysis

Projeto desenvolvido para a disciplina de Ciência de Dados e Inteligência Artificial, com o objetivo de explorar e analisar o conjunto de dados Students Performance in Exams (Kaggle).

O script realiza a carga do dataset, criação de colunas derivadas e cálculo de estatísticas descritivas (média, mediana, desvio-padrão, valores mínimos/máximos e moda) para as principais variáveis relacionadas ao desempenho dos alunos.

Ferramentas utilizadas:

Python (pandas, numpy)

Orange Data Mining (para visualização e modelagem)

PyCharm (ambiente de desenvolvimento)

Etapas do projeto:

Importação e exploração do dataset.

Criação das variáveis derivadas average_score e performance_level.

Cálculo das estatísticas descritivas para 10 colunas principais.

Geração do arquivo resumo_estatistico.csv contendo as métricas exigidas pelo projeto.

Objetivo final:
Identificar fatores que influenciam o desempenho escolar e demonstrar o uso de ferramentas de ciência de dados para extração de insights educacionais.

Dataset original: Students Performance in Exams – Kaggle
